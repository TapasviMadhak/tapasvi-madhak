# TM Portfolio V2

A vibrant neobrutalist cybersecurity portfolio built with React, Vite, Tailwind CSS, and Framer Motion.

## Features

- Live GitHub projects section (auto-updates from GitHub repos)
- README-based project descriptions
- Skills and certifications dashboard with elegant scroll panels
- Custom neobrutalist smooth cursor interactions
- Medium blog feed integration

## Tech Stack

- React
- Vite
- Tailwind CSS
- Framer Motion
- Lucide React

## Local Development

1. Install dependencies:

```bash
npm install
```

2. Run development server:

```bash
npm run dev
```

3. Build for production:

```bash
npm run build
```

4. Preview production build:

```bash
npm run preview
```

## Configuration Notes

- GitHub repos are fetched from username configured in `src/data/portfolioData.js`.
- Medium posts are fetched from the RSS feed configured in `src/App.jsx`.
